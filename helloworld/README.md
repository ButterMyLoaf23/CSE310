# Hello World

Simple program written in python for Hello World.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Python 3 on your pc
2. Open the project folder in vscode or another IDE
3. open the terminal
4. run the program using python helloWorld.py

Instructions for using the software:

1. Run the program with the command python helloWorld.py
2. The program will run and display "Hello World!"
3. You can change the message to say whatever you want.
